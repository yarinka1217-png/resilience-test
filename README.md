# Resilience Psychological Test — Built with HTML, CSS and JavaScript

This is an interactive psychological test that evaluates personal resilience based on 16 carefully designed questions.  
It uses dynamic JS rendering, a progress bar, localStorage scoring, and generates a personalized final result.

## Features
- 16-question adaptive test  
- Dynamic rendering (no page reload between questions)  
- Progress bar with smooth animation  
- Result calculation via localStorage  
- 5 personality-based resilience types  
- Clean, modern UI  
- Mobile responsive  

## Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript
